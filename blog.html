Question-1: What are the different ways to select an element in the DOM?

There is several ways to select and element .
Some of them mentioned below :

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, and textContent is given below :

  • innerHTML : It returns the HTML content of an element.
  • innerText : It returns the text content inside an element.
  • textContent : It returns all the HTML tags and the text inside the HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique involving adding event listeners to a parent element instead of adding them to the child elements.
The technique helps us when we have a lot of elements handled in a similar way, then instead of assigning a handler to each of them , we put a single handler on their common parrent.

Question-4: What is event bubbling in the DOM?

Event bubbling is a system in the DOM event propagation model where an event first triggers on the target element and then propagates up through its parent elements until it reaches the head of the document.

Question-5: How do you create, add, and remove elements using JavaScript?

  1. Create an element : createElement(tagname) which create a tag
  2. Add an element : appendChild(childElment) which appends childElement to parent Element
  3. Remove an element : removeChild(childElment) which remove the childElement from the parent Element
